# CampusForum 校园论坛前端项目

## 项目概述与功能说明

CampusForum是一个为大学生设计的校园论坛平台，旨在为学生提供一个交流、分享和获取信息的空间。

### 核心功能

- **用户系统**：注册、登录、个人资料管理
- **帖子管理**：发布、编辑、删除帖子，支持分类和标签
- **互动功能**：评论、点赞、收藏帖子
- **消息通知**：接收评论、点赞等通知
- **校园风采**：轮播展示校园图片
- **搜索功能**：支持按关键词搜索帖子

### 页面结构

- **首页**：展示最新帖子、热门标签
- **帖子详情**：查看完整帖子内容和评论
- **发布帖子**：创建新帖子
- **个人中心**：管理个人信息和发布的帖子
- **消息中心**：查看通知
- **发现页面**：校园风采轮播展示

## 技术栈与依赖说明

### 前端技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript (ES6+)**：交互逻辑
- **LocalStorage**：本地数据存储（开发环境）

### 外部依赖

- 无第三方库依赖，使用原生JavaScript实现所有功能
- 响应式设计，适配不同设备屏幕

## 本地运行与构建步骤

### 运行环境

- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 无需服务器环境，可直接打开HTML文件运行

### 运行步骤

1. **克隆项目**
   ```bash
   git clone <仓库地址>
   cd CampusForum
   ```

2. **直接运行**
   - 打开 `Html/index.html` 文件即可访问首页
   - 其他页面可通过导航栏或直接访问对应HTML文件

3. **开发模式**
   - 修改 `JavaScript/api.js` 中的 `useLocalStorage` 变量控制是否使用本地存储
   - 开发环境默认使用LocalStorage模拟数据，无需后端服务

### 构建与部署

- 本项目为纯前端项目，无需构建步骤
- 部署时只需将整个项目目录上传到服务器即可

## 已知问题与待解决事项

### 已知问题

1. **数据存储**：当前使用LocalStorage存储数据，刷新页面后数据会保留，但清除浏览器数据后会丢失
2. **图片上传**：暂时不支持图片上传功能，仅使用静态图片
3. **响应式设计**：在某些极端屏幕尺寸下可能存在布局问题
4. **性能优化**：大数据量时可能存在渲染性能问题

### 待解决事项

1. **后端集成**：替换LocalStorage为真实后端API
2. **图片上传**：实现图片上传功能
3. **性能优化**：实现虚拟滚动，提升大数据量渲染性能
4. **安全性**：增强前端安全措施，防止XSS等攻击
5. **国际化**：支持多语言切换

## 后端API对接说明

- 前端已按照 `docs/api-spec.md` 中的接口规范实现了Mock数据
- 后端开发完成后，只需修改 `JavaScript/api.js` 中的接口调用部分即可
- 具体接口规范请参考 `docs/api-spec.md` 文档
