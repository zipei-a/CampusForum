# 校园论坛用户操作手册

## 1. 系统介绍

校园论坛是一个为在校学生提供交流、分享和学习的在线平台，用户可以在平台上发布帖子、参与讨论、关注感兴趣的话题和用户。

### 1.1 主要功能

- **用户认证**：注册、登录、退出登录
- **帖子管理**：发布、浏览、搜索、评论、点赞、收藏帖子
- **个人中心**：查看和编辑个人资料、管理发布的帖子和收藏
- **消息通知**：接收系统通知和评论回复
- **分类浏览**：按分类查看帖子
- **标签系统**：通过标签筛选相关帖子

## 2. 注册与登录

### 2.1 注册流程

1. 打开校园论坛首页
2. 点击右上角的「注册」按钮
3. 填写注册信息：
   - 用户名（3-20个字符）
   - 邮箱（可选，用于找回密码）
   - 密码（6-20个字符）
   - 确认密码
4. 阅读并同意服务条款和隐私政策
5. 点击「注册」按钮完成注册
6. 注册成功后自动登录并跳转至首页

### 2.2 登录流程

1. 打开校园论坛首页
2. 点击右上角的「登录」按钮
3. 输入用户名和密码
4. 点击「登录」按钮完成登录
5. 登录成功后跳转至首页

### 2.3 退出登录

1. 登录状态下，点击右上角的用户头像
2. 点击「退出」按钮
3. 确认退出后，系统会自动跳转到首页

## 3. 帖子管理

### 3.1 发布帖子

1. 点击顶部导航栏的「发布」按钮
2. 填写帖子信息：
   - 标题（最多100个字符）
   - 内容（最多10000个字符）
   - 选择分类
   - 添加标签（多个标签用逗号分隔）
3. 点击「发布」按钮
4. 发布成功后自动跳转到帖子详情页

### 3.2 浏览帖子

1. 首页会显示最新发布的帖子
2. 可以通过顶部导航栏的分类标签筛选不同分类的帖子
3. 点击帖子标题进入帖子详情页

### 3.3 搜索帖子

1. 在顶部搜索框中输入关键词
2. 点击搜索按钮或按回车键
3. 系统会显示相关的搜索结果

### 3.4 评论帖子

1. 进入帖子详情页
2. 在评论区输入评论内容
3. 点击「发表评论」按钮
4. 评论会显示在评论列表中

### 3.5 点赞和收藏帖子

1. 进入帖子详情页
2. 点击帖子下方的点赞图标可以点赞/取消点赞
3. 点击帖子下方的收藏图标可以收藏/取消收藏

## 4. 个人中心

### 4.1 查看个人资料

1. 点击右上角的用户头像
2. 选择「个人中心」进入个人资料页面
3. 个人资料页面会显示用户名、个人简介、发布的帖子数、收藏的帖子数和获得的点赞数

### 4.2 管理发布的帖子

1. 进入个人中心页面
2. 点击「我的帖子」标签
3. 这里会显示您发布的所有帖子
4. 点击帖子标题可以进入帖子详情页

### 4.3 管理收藏的帖子

1. 进入个人中心页面
2. 点击「我的收藏」标签
3. 这里会显示您收藏的所有帖子
4. 点击帖子标题可以进入帖子详情页

### 4.4 编辑个人资料

1. 进入个人中心页面
2. 点击「设置」标签
3. 可以编辑个人简介和邮箱
4. 点击「保存修改」按钮完成更新

## 5. 消息通知

### 5.1 查看通知

1. 点击顶部导航栏的「消息」按钮
2. 消息页面会显示系统通知和评论回复
3. 未读通知会显示为蓝色

### 5.2 标记通知为已读

1. 进入消息页面
2. 点击通知旁边的「标为已读」按钮
3. 或点击「全部标记为已读」按钮标记所有通知为已读

## 6. 分类与标签

### 6.1 按分类浏览

1. 顶部导航栏会显示不同的分类标签
2. 点击分类标签可以查看该分类下的所有帖子

### 6.2 按标签筛选

1. 帖子详情页会显示帖子的标签
2. 点击标签可以查看含有该标签的所有帖子
3. 首页右侧会显示热门标签，点击可以查看相关帖子

## 7. 常见问题解答

### 7.1 忘记密码怎么办？

目前系统暂不支持密码找回功能，请妥善保管您的密码。

### 7.2 如何修改用户名？

用户名一旦注册后不可修改，请谨慎选择用户名。

### 7.3 帖子发布后可以修改吗？

目前系统暂不支持编辑已发布的帖子，请在发布前仔细检查内容。

### 7.4 如何删除自己发布的帖子？

目前系统暂不支持删除帖子功能，如有需要请联系管理员。

### 7.5 可以上传图片吗？

目前系统暂不支持图片上传功能，仅支持文本内容。
